Quickstart

For experienced people familiar with using Jekyll, the Quickstart section may help start by creating a Starter Web - the build-in website for J1. This article gives an overview of what software is needed and how to run the base (web development) commands for:

  • creating a J1 project

  • setting up your run-time

  • (re-)build a web

  • run a web

Setting up a J1 project and creating the first website using J1 will take about 15 minutes. But, you should know what you doing!
