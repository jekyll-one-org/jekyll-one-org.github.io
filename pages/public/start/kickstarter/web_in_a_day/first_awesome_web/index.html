    Recap

    OAuth2 is an authentication framework, a protocol suite that enables applications to obtain (limited) access to user account data hosted at service providers.

    The protocol suite introduces four roles, discussed in section [OAuth Roles], that interacts each other in a sequence, a flow. In section [Base OAuth Flow] the sequence was discussed in more detail.

    High Level OAuth Protocol Flow

    Keep in mind the roles and the fact, the OAuth framework introduce a sequence, a flow. All this is theory - yes for sure, but good to know.
