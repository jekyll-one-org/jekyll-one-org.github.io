J1 Theme supports popular icons font sets out-of-the-box. The icon sets Material Design Icons MDI and FontAwesome FA are exellent in design and have a rich set of icons for many categories typically used on web pages.

5-10 Minutes to read

The J1 Icon Picker supports the following icon sets:

  • Material Design Light Icons MDIL, 260+ icons of version 0 (February 2023)

  • Material Design Regular Icons MDI, 7200+ icons of version 7 (March 2023)

  • FontAwesome Icons FA, 2000+ icons of version 6 (March 2023)

The free version of FontAwesome FA does not support the full icon set available. The commercial Pro Version supports 26000+ icons but requires a paid subscription plan.

Select the icon set from the dropdown below. To show the selected icons, click on the button Show Icons Selected.

Show Icons Selected

A double-click on an icon copies the selected icon class data to the clipboard. This data can be easily used in the configuration file of J1 Template.

Note that Base MDI Icons is a subset of about 2600+ icons taken out of the full set provided by the regular icon set MDI. Because the regular MDI icon set is very huge, 7200+ icons, for performance reasons it is not recommended to load all icons while a page is initialized.
