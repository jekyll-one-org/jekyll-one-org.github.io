Jekyll One

featured template for jekyll

Jekyll One Theme is a Gem-based Theme made for the Static Site Generator (SSG) Jekyll. J1 Theme meets Jekyll and Bootstrap. And makes a lot of friends. The template system combines the best of free software for the web. Jekyll One Theme is OpenSource and the modules included are free to use as well.

No pain for private or professional purposes.
