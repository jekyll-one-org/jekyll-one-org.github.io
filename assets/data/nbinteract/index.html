This website uses the free service Binder to process interactive widgets from content containing textbooks (HTML exported Jupyter Notebooks). The Binder Service allows you to create custom computing environments that can be shared and used by many remote users. A Binder service is powered by BinderHub Repo on Github, an open-source tool that runs on Kubernetes, a portable, extensible, open-source platform for managing containerized services. One such deployment lives at Binder Home, and is free to use.

Each interactive element presented on this page uses time-consuming operations that take a while to finish. The images and the interactive elements are built through a backend in the cloud. Please be patient to see the results.
