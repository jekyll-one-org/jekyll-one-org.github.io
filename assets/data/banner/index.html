Go the static way

powerful modern static webs

If you’re not familiar with Jekyll (or other website generators), Jekyll is a so-called static site generator. Think of it as a modern but lightweight Content Management System (CMS) - like Wordpress or Joomla but much, much simpler and without all the complexity.

For many use cases, there is no need to use a full blown CMS to manage a website. If you go the static way, no database is needed and no PHP-based dynamic rendering engine that slows down the performace of your website. For e.g. document servers like Bootstrap or Docker, Jekyll is used for good reasons.
All you’re doing at the end of the day is writing your content and the builtin engines does all the tricky things in behind needed for a website: design pages, putting out HTML, CSS and Javascript. As a result, the site creation process gets out of your way. It’s done by J1 Template and allows you to concentrate on what truly matters: your content.
