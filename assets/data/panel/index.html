People quite often ask if static websites make sense for anything beyond a one-page website. The answer is clearly yes!

Static websites can make sense for any type of website, from a portfolio website to an e-commerce store. Static websites are becoming increasingly popular because they are faster, more secure, and easier to maintain than traditional dynamic websites. Static websites are also great for SEO (search engine optimization) as they are easier for search engines to crawl and index.

People quite often ask if static websites make sense for anything beyond a one-page website. Some others see in static websites the stone age of the World Wide Web. What's about static websites today? What's achievable? This board highlights top features, which make static webs not only an option but a state-of-the-art technology to create fast, secure, and sophisticated websites for many use cases.

  • Jekyll One
    General Features

    Current software counts. Excellent tools in their latest versions make the rocket engine for each and every website. Jekyll One is using current software in their latest versions. All of them are OpenSource and free to use. No license issues for private or professional use.

    The base of J1.

    Platforms

    The theme J1 Theme is based on the Ruby language of version 2.x. For sure, not the latest language version for Ruby but the supported platform for the latest Jekyll software.

    Websites created by Jekyll One Theme are projects managed by NodeJS. The built-in script engine provided by the package manager npm of NodeJS simplifies the management of J1 projects a lot. Well-known commands typically used to manage software projects are at your fingertips. Easy to learn, easy to remember. Commands support creators to concentrate on the content.

    Not get lost in project management.

    Ruby 2.7
    Jekyll 4.3
    NodeJS 14.x

    Frameworks

    To not reinvent wheels, J1 Theme is based on ready-to-use frameworks, extensions, and modules. All build-in frameworks are industry standard, well known, and accepted worldwide.

    For current webs, a must: responsiveness and modern design. Jekyll One is based on Bootstrap, the leading platform to create today's responsive webs.

    Professional websites on the internet define standards. All current sites support dynamic, fast loading, and some more features. For fastest loading, all HTML, CSS, and Javascript have been compressed automatically for production. Going further for dynamic, AJAX loading for J1 webs is supported as well.

    No way: icons and icon fonts are required to make a site understandable. J1 supports four icon fonts to manage this. Icons are used for extensions, modules, and content.

    Software products of today are not easy to manage. To get the complexity out of your way, J1 is fully configurable. No need to dive in the deep.

  • J1 Theme

    J1 Template is a Bootstrap V5 website template for the static site generator Jekyll. The template system J1 was initially developed to create a simple but fully configurable creation tool for document servers. Document servers provide websites, or other digital content like PDF files, to publish the project documentation to the public.
